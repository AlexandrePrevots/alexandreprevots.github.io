Musée des Arts Forains

UX / UI design




During my UX design master's degree, I worked on a project to redesign the website of the Musée des Arts Forains. With my team, we brainstormed a solution using the UX and web methodology to design a prototype in mobile first.

We analyzed user needs to find out how to make them live an unforgettable experience in the museum, both in real and virtual.

You can see above the deliverable in pdf.

Here is the methodology we've used :

• Audit

• Benchmark

• User search

• Site mapping

• Wireframes

• Moodboard

• Storyboard

• Brand redesign

• Interface design

• Service design
