Matmut

UX / UI design / Front-end web development



I worked for the redesign of matmut.fr website and redesign the different customer journeys (quotes and subscriptions).

I designed on mobile first to make the website accessible on all devices and evolve with the web usages.

The first part of each projects was to sketch the wireframes (paper first and then on Axure) to test them in user tests. Users tests raised issues and helped optimize wireframes for re-testing. Then, I worked on graphic design part.

To make a fully responsive website, I integrated various animations in html 5, which were previously in Adobe Flash.





I also worked on the redesign of the home screen of the application "Ma Matmut" and on a feature so that the user can customize their interface. The interfaces have been tested during users tests to see if interfaces and features are intuitive.

To support the user, I designed and developed (in html/css) a walkthrough. The walkthrough only appears during the first user connection and lets him know the main features present on the application.


I am currently working on innovative projects to bring the company into digital trends.
